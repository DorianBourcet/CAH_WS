Protocole de communication disponible par le membre:

-START:	En écrivant "START" vous signalé le désir de démarrer une partie

-PROPOSITION: En écrivant "PROPOSITION" suivis du/des numéro(s) de carte(s) séparé par un espace (Exemple: PROPOSITION 2) vous signalé le désir 
	      de jouer la carte avec le numéro suite à la proposition. Pour plus qu'une carte veuillez séparé le numéro des cartes par un espace
	      (Exemple: PROPOSITION 3 4).

-VOTE: En écrivant "VOTE" suivis du numéro de la proposition choisie (Exemple: VOTE 1) vous signalé le désir de choisir la proposition avec le numéro après lacommande VOTE.

-CHAT: En écrivant "CHAT" suivis d'un message (Exemple: CHAT Hey comment ca va?) vous envoyez un message aux autres joueurs.


Protocole de communication non-disponible par le membre (protocole pour la fonctionnalité de l'application):

-LOGIN: permet l'authentification au serveur.

-LOGOUT: permet la déconnection au serveur.

-FORGOT: permet l'envoie de mot de passe d'un user associé au mail spécifier.

-SUBSCRIBE: permet l'inscription d'un nouveau membre dans la base de donnée.
